# ระบบรายงานโครงการโรงเรียนบ้านจิก

เว็บแอปต้นแบบภาษาไทยสำหรับครูโรงเรียนบ้านจิก ใช้กรอก บันทึก และพิมพ์รายงานโครงการได้จากเว็บเบราว์เซอร์ โดยไม่ต้องติดตั้งฐานข้อมูลหรือเครื่องมือพัฒนาเพิ่มเติม

## ความสามารถของต้นแบบ

- กรอกข้อมูลพื้นฐานโครงการ เช่น ชื่อโครงการ ปีการศึกษา ผู้รับผิดชอบ หน่วยงาน และระยะเวลาดำเนินงาน
- บันทึกหลักการและเหตุผล วัตถุประสงค์ และกลุ่มเป้าหมาย
- เพิ่มกิจกรรมและวันที่ดำเนินงานได้หลายรายการ
- เพิ่มงบประมาณและรายการค่าใช้จ่าย พร้อมคำนวณยอดรวมอัตโนมัติ
- กรอกผลการดำเนินงาน ตัวชี้วัด ปัญหา อุปสรรค และข้อเสนอแนะ
- แนบรูปภาพหลักฐานได้หลายภาพ
- บันทึกหลายโครงการในเครื่องเดิมด้วย Local Storage ของเบราว์เซอร์
- แสดงตัวอย่างรายงานและสั่งพิมพ์ได้ทันที

## วิธีเปิดใช้งาน

### วิธีที่ง่ายที่สุด

1. ดาวน์โหลดหรือเปิดโฟลเดอร์โปรเจกต์นี้
2. ดับเบิลคลิกไฟล์ `index.html`
3. ทดลองกรอกข้อมูลและกด **บันทึกโครงการ**
4. กด **พิมพ์รายงาน** เมื่อต้องการพิมพ์หรือบันทึกเป็น PDF

### วิธีเปิดผ่านเว็บเซิร์ฟเวอร์ในเครื่อง

หากมี Python ติดตั้งอยู่ สามารถเปิดผ่านเว็บเซิร์ฟเวอร์อย่างง่ายได้ดังนี้

```bash
python3 -m http.server 8000
```

จากนั้นเปิดเบราว์เซอร์ไปที่

```text
http://localhost:8000
```

## หมายเหตุการใช้งาน

ข้อมูลในต้นแบบนี้ถูกเก็บใน Local Storage ของเบราว์เซอร์ จึงเหมาะสำหรับทดลองใช้งานหรือใช้งานในเครื่องเดียว หากล้างข้อมูลเบราว์เซอร์หรือเปลี่ยนเครื่อง ข้อมูลอาจหายได้ ในเวอร์ชันถัดไปสามารถพัฒนาให้มีระบบผู้ใช้ ฐานข้อมูลกลาง และการส่งออกรายงานเป็นไฟล์เอกสารได้

## ไฟล์หลักของระบบ

- `index.html` โครงสร้างหน้าเว็บและแบบฟอร์มรายงานโครงการ
- `styles.css` รูปแบบหน้าจอและรูปแบบสำหรับการพิมพ์
- `app.js` การบันทึกข้อมูลหลายโครงการ การแนบรูปภาพ การคำนวณงบประมาณ และการสร้างตัวอย่างรายงาน
